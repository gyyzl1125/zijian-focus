# 自见

一个专注 + 任务 + 时间轴 + 设置的安卓应用原型。当前先用浏览器确认体验，后续可用 Capacitor 打包成 APK。

## 已有功能

- 四个独立界面：专注、任务、时间轴、设置
- 专注计时：5 到 180 分钟自由选择
- 小鸡草地场景：天气随机变化、云漂移、种子飘落、草生长、小鸡随机走动
- 专注过程中小鸡会从破壳成长到长成
- 任务可设置开始、结束、提醒方式、任务颜色
- 提醒支持提前几分钟/几小时/一天，也支持指定详细提醒时间
- 时间轴按日期展示任务和专注记录
- 设置页包含热力图和桌面小组件预览
- 计时会保存真实结束时间，息屏后再打开会按真实时间恢复或结算

## 电脑上运行

双击：

```text
start-dev.bat
```

保持弹出的窗口不要关闭，然后电脑浏览器打开：

```text
http://127.0.0.1:5174/index.html?v=8
```

如果仍然看到旧界面，先打开一次：

```text
http://127.0.0.1:5174/clear-cache.html
```

## 安卓手机预览

电脑和手机连接同一个 Wi-Fi。启动 `start-dev.bat` 后，用手机浏览器打开电脑的 Wi-Fi IPv4 地址，例如：

```text
http://10.194.231.33:5174/index.html?v=8
```

如果手机打不开，通常是 Windows 防火墙拦截了 Python 的网络访问，需要允许 Python 通过专用网络。

## 关于安卓小组件和黑屏计时

当前浏览器原型只能预览桌面小组件的内容样式，不能真正添加到安卓桌面。真正的桌面 Widget、锁屏/后台稳定提醒，需要打包 APK 后接入安卓原生 Widget 和本地通知。

计时准确性已先做基础保障：开始专注时会保存真实结束时间，息屏或浏览器暂停后再打开，会按真实时间恢复或结算。
